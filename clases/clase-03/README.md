# Clase 03 - Configuración de Servidor Local y CV HTML

1. Instale XAMPP en su computador
2. Configurar Apache para que utilice el puerto 8080
3. Cree su CV en HTML (puede utilizar AI ) y lo guarda como index.html en el Document Root
4. Ingrese a su navegador: http://localhost:8080
5. Deberá mostrar su CV en HTML.
